## wooltube
# wooltubeの公式リポジトリです。
## どんどんデプロイしてください!!
### デプロイ方法
1. [Render](https://render.com/) にログインまたは新規登録
2. 「New」 → 「static site」 を選択
3. RemixしたWOOLtubeのリポジトリの公開URLを入力
4. 一番下の完了ボタン的なとこを押す
5. 数分待つと完了です
### html.cafe
1. [html.cafe](https://html.cafe)にアクセス
2. [コード](https://raw.githubusercontent.com/woolisbest/WOOLtube/refs/heads/main/html-cafe.html.txt)を張り付ける
3. 終わり
